# Megaline---Analise-Estatistica-e-Exploratoria-
Limpeza e tratamento de dados (remoção de duplicados e preenchimento de valores ausentes) em dataset de mais de 4,5 milhões de registros. Aplicação de EDA com Pandas, NumPy e Matplotlib para identificar padrões de compra, como pico de pedidos entre 10h–15h e produtos mais recorrentes.
